 Forensic DNA Profiling System

Overview
This project is a bioinformatics-based application that simulates forensic DNA analysis using Short Tandem Repeat (STR) markers. It compares DNA profiles and determines the likelihood of a match between samples.

---

Objective
- To analyze DNA profiles using STR loci
- To compare genetic markers between individuals
- To calculate match percentage
- To generate a forensic-style analysis report

---

## How to Run

### Method 1: Google Colab
1. Open the notebook in Google Colab
2. Upload `dna_profiles.csv`
3. Run all cells

### Method 2: VS Code
1. Install dependencies:
   pip install pandas
2. Run:
   python main.py
